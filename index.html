Projects

Living Quarters

Living Quarters was made to be an alternative to the dozens of apps you would need to use to track all of the things that roommates have to deal with. We decided that it makes sense for this to be a mobile app, so it was built thinking the only people who would use this app were people on their phone.

Open-Source Data Visualization

I constructed a new model for an open-source data visualization library built on D3. The model integrated 4 data types on 3 axes, allowed for dynamic data selection, and zooming on sets of data. This is being used by mechanical engineers fighting global warming.
